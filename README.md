# EICHE Malerbetrieb Lumora Test

Generated Lumora test website for EICHE Malerbetrieb GmbH.

## Local Preview

```bash
npm install
npm run dev
```

Then open the local Vite URL shown in the terminal.

## Notes

- The site is a Vite static frontend.
- It uses client-side routing for pages such as `/leistungen`, `/team`, and `/kontakt`.
- Generated source is intentionally separate from the `LumoraSkill` repo.
